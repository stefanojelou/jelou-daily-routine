## 🆕 New (2)

### 🆕 Mobile signups are dead-on-arrival — 2.9% activate vs 27% desktop

55% of signups (4,536/8,084) arrive on mobile, but cohort-by-signup-device only 2.9% build (node_used, any device, within 14d) vs desktop 26.9%. The builder canvas is desktop-only, so most signups structurally can't activate on the device they signed up on.

- **Metric:** **mobile 14d build-activation = 2.9%**
- **Sources:** mixpanel
- **Confidence:** high (N=4,536 mobile signups; cohort-by-signup-device, any-device node_used, 14d window)
- **Caveat:** node_used is a desktop-only drag action (31,722 desktop vs 128 mobile firings); recent signups (<14d) have a partial window, mildly understating.

### 🆕 Media acquisition is 77% mobile → activates at ~half of organic

Media signups (utm_source present, 1,695/8,084 = 21%) are 77% mobile vs ~50% for organic. Because mobile rarely activates, media converts to real building at ~6% vs ~15% organic — media spend is largely buying dead-on-arrival mobile signups.

- **Metric:** **media 14d build-activation = ~6%**
- **Sources:** mixpanel
- **Confidence:** medium-high (media N=1,695; organic baseline N=6,389)
- **Caveat:** utm_source on ~21% of signups; ~6% is event-device-based and likely modestly understated (~8% cohort-corrected). Reconciled revenue/conversion needs Stripe (deferred).
